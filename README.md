# Web Traffic Analytics - Task 3

## Project Overview
Examine website traffic data to understand user behavior.
Identify key metrics such as page views, session duration,
and drop-off points to improve engagement.

## Analysis Performed
1. Traffic overview (sessions, bounce rate, conversion rate)
2. Most visited pages identified
3. Avg time spent per page calculated
4. Traffic source analysis (Google, Email, Social, Direct)
5. Device usage breakdown (Mobile, Desktop, Tablet)
6. Drop-off points identified
7. Key insights and recommendations provided

## Key Findings
- Products page has the most views
- Pricing page has highest avg session time
- Google is the top traffic source
- Mobile users are highest — mobile-friendly design needed
- Email campaigns have best conversion rate

- traffic.py → Analysis script
- traffic_analysis.csv → Output dataset
